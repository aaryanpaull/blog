CivicLens — AI-Powered Civic Issue Reporting

Short overview: CivicLens is an AI-driven civic complaint reporting and routing system that empowers citizens to report civic issues (potholes, garbage, sewage, streetlights, waterlogging) using photo + description + location. The system validates reports with AI, assigns severity (1–10), tags keywords, and routes issues to the appropriate government departments with step-by-step action suggestions.

Deployed Prototype

https://civiclens-five.vercel.app/

Features

  • Photo upload (Sharp compression)
  • Automatic location tagging (GPS)
  • AI verification (spam filtering) via Gemini 2.5 Flash
  • Severity rating (1–10) and priority assignment
  • Multi-department routing (Chennai mapping included)
  • Step-by-step actionable suggestions for authorities
  • Interactive complaint map & personal dashboard
  • Leaderboards, certificates, and gamification
  • Multi-language support (EN, HI, TA, TE, KN, GU, BN)

Tech Stack

Backend: Node.js + Express
AI: Google Gemini 2.5 Flash
Image Processing: Sharp (WebP compression)
Frontend: Static HTML/CSS/JS
Storage: Optional Firebase / MongoDB

Getting Started

  1. Clone the repo
  2. npm install
  3. Create a .env with GEMINI_API_KEY and PORT
  4. npm start

Guide to Machine Learning as a Beginner

📅 October 2025 • 5 min read

Machine Learning (ML) is everywhere — from Netflix recommendations, spam filters, Google Maps predictions, to self-driving cars. AI isn't just a buzzword anymore, its implications, that is Machine Learning is shaping every crucial process, and we aren't even aware of it fully.

This blog will be essential for you to know how to start learning Machine Learning, even if you have zero prior experience.

What Is Machine Learning?

Machine learning is the subset of artificial intelligence (AI) focused on algorithms that can "learn" the patterns of training data and, subsequently, make accurate inferences about new data. This pattern recognition ability enables machine learning models to make decisions or predictions without explicit, hard-coded instructions.

Instead of writing rules manually, we provide data and algorithms learn patterns.

Examples include:

  • Gmail spam filter - Detects unwanted emails
  • YouTube recommendations - Suggests videos you may like
  • Medical diagnosis systems - Predict diseases from scans
  • Self-driving cars - Detects objects, predicts movement

Understanding the Core workflow of ML:

Every ML project follows a similar flow:

  1. Define the problem
  2. Collect data (emails)
  3. Clean and prepare data
  4. Choose a model (eg: logistic regression, neural networks)
  5. Train the model (feed data to learn patterns)
  6. Test & evaluate performance
  7. Deploy

Skills You Need to Start ML

Programming:

Preferred language: Python

  • Simple syntax
  • Huge ML ecosystem

Learn:

  • Variables, loops, functions
  • Data structures
  • File handling

Tools: Jupyter Notebook / Google Colab

Math Fundamentals-

  • Linear algebra - Working with vectors & matrices
  • Statistics & Probability - Understanding data & predictions
  • Calculus (basics) - Understanding how models learn

Data Handling-

Learn to work with data using:

  • Pandas - Data manipulation
  • NumPy - Numerical computations
  • Matplotlib / Seaborn - Data visualization

Beginner Friendly ML Libraries to Learn

  • scikit-learn |
  • TensorFlow / PyTorch - For deep learning (later stage)

Hands-On Project Ideas

  • Predict house prices - Regression
  • Classify flowers (Iris dataset) - Classification
  • Spam email detection- Natural Language Processing
  • Handwritten digit recognition (MNIST) - Neural networks

Tip: Start with scikit-learn, then move towards deep learning, which is further advanced.

90-Day Roadmap-

Weeks Focus
1–2 Python basics + Numpy, Pandas
3–5 Statistics + Exploratory Data Analysis
6–8 ML algorithms using scikit-learn
9–12 Mini projects + Kaggle challenges

Resources-

Courses:

  • Google — Machine Learning Crash Course
  • Coursera — Andrew Ng ML Course
  • Kaggle — Free practical ML courses
  • Youtube - YT Channel named "Bro Code"

Practice Platforms-

  • Kaggle (learn already made models and study them elaborately)
  • Hackerrank (Machine Learning Section)

Conclusion

Machine Learning is one of the most exciting and impactful fields today. With the right roadmap, mindset, and consistent practice, anyone can learn it — including you.

Why C++ is still relevant in today's world

📅 October 2025 • 4 min read

When looking at the computer programming landscape, it's hard not to notice that C++ is nearly half a century old, and yet it's not fading away, but rather continuing to be a cornerstone in the world of computer science.

Well-known in the world of research, high-performance applications, and certain industries, the popularity of C++ is based not on tradition but on its ability to effectively address computing problems that newer languages can't, and still hold a balance between the power and speed of low-level systems and the structure of higher-level languages.

Built with the specific task in mind to deliver efficient computing resources with high-level construction features, the recent standards improvements to C++ have supplied memory shielding, lambda features, and better concurrent processing, plus a far more expansive standard library which make it more adaptable and secure than its previous versions.

At the heart of its reason for being is performance. When applications need to be lightning fast, whether it's for operating systems, embedded systems, simulations, numerical computations or games, C++ stands out from the rest. The likes of Google, Microsoft, Adobe and companies in the gaming and finance industries have made C++ the go-to language for their performance-critical components, showcasing its value.

C++ is also the top choice for competitive programmers and academics, and websites like Codeforces and the ICPC encourage its use. Coming hotfooting through the world of computer programming, C++ gives students a firm foundation in memory management, data structures and mental algebra, turning them into top-notch problem solvers.

It's not always the best choice for prototyping or web development, but C++ is the only game in town when high performance, precision and control are a priority.
